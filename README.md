# c-music-playlist

A simple terminal-based music playlist manager written in C. This project demonstrates the use of structures, pointers, dynamic memory allocation, and file organization in C.

## 📋 Features

- Add songs to playlist
- Delete songs
- Display current playlist
- Play next/previous song
- Search songs by name
- Menu-based user interface in terminal

## 🛠️ Technologies

- C Programming Language
- GCC Compiler
- Works on Mac/Linux/Windows (with gcc or MinGW)
